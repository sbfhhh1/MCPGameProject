# 青铜典藏 UI 复刻设计规范（学习自原视频）

## 一、整体设计语言
- **背景**：暖黑炭灰纸纹（#1a1815 ~ #232020），常叠加水墨泼墨笔触
- **主色**：金色 #c9a449 / 亮金 #e6c46a（标题、装饰线）
- **辅色**：朱红印章 #9a2018；正文暖灰白 #c9c3b8
- **字体**：
  - 标题 = 毛笔书法/篆书（粗、有力、飞白）
  - 正文 = 楷书（清秀、有顿挫）
  - 英文 = 衬线大写、字间距很宽（letter-spacing 大）
- **装饰**：祥云金线（回纹/云纹）、朱红印章、金色细分隔线

## 二、顶部导航（所有页通用）
- 左上角：金色方印 logo（繁复篆字，红黑金）
- 居中偏右：起源 夏 商 西周 春秋 战国 秦汉
  - 字间距宽，当前页金色高亮，其余暗金
  - 字号约 18-20

## 三、文物详情页模板（重点先做）
布局（1920×1080 基准）：
- **左侧文字区**（x: 80~720）：
  - 金色书法**文物名**（大，如"青銅鏡""銅爵""曾侯乙編鐘"），字号 48-56
  - 下方英文小标（大写宽字距，如 BRONZE MIRROR），字号 13，暗金
  - 下方**正文**（楷书，暖灰白），首行缩进，行高 1.7，字号 19-21
- **右侧文物图**（x: 1000~1500，垂直居中）：
  - 无白底，融入暗背景
  - 后面有水墨泼墨笔触
- **最右"国宝档案"竖排**（x: 1650~1850）：
  - 国/宝/档/案 四字金色书法，**错落排布**（国在右上、宝右下、档左下、案…交错）
  - 左侧竖排小字"青铜时代"（暗金，小）
  - 下方朱红方印
- **祥云金线**：右下或左下角，金色回纹云线

## 四、朝代封面页模板（后续做）
- 巨型金色书法朝代名（夏代/商/西周…，竖排或横排各页不同）
- 竖排小字"青铜鼎盛时期" + 朝代纪元（公元前…）
- 英文朝代名（XIA DYNASTY…，竖排大写宽字距）
- 朱红印章
- "查看详情"金色胶囊按钮
- 每页独特装饰：夏=水墨、商=群山诗句、西周=祥云回纹、春秋=L框线、战国=大圆环+山、秦汉=水墨荷

## 五、当前实现 vs 原作 差距（待修复）
1. 标题用黑体 → 应换书法字体
2. 文物图带白底 + 位置压字 → 去白底、移到右侧不压字
3. 缺祥云金线装饰
4. "国宝档案"是规整单列 → 应错落书法排布
5. 印章 logo 太简 → 应繁复篆字红金方印
6. 缺水墨泼墨笔触背景
